JSAdmin is a admin. It base on NodeJS + adminLTE  + VUE + Keystone.

https://github.com/almasaeed2010/AdminLTE

#Dependencies
AdminLTE
NodeJS
express
Vue
Keystone->mongodb
form-create(vue+iview)

#install
npm install
npm install webpack -g
npm run build
node index.js

# browser http://127.0.0.1:3000/admin/index.html
